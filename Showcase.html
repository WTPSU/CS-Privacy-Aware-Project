Spreading Awareness for Local Differential Privacy (LDP)

By Westley T and Hong

A Brief Introduction

Differential privacy (DP) is a technique for preserving individual privacy. It does this by adding controllable noise to the data so that one person's data does not change the output significantly, making it harder for attackers to gain information.

Central differential privacy is a type of DP that is already widely used, Its application is mostly on the server/host/creator side with raw data. Local differential privacy (LDP) on the other hand, is less common and could provide more privacy if it was widely adapted more. LDP is applied on the client side before reaching the server, ensuring the data given is "accurate" enough to be useful while also preserving some privacy.

The goal of this website is to spread awareness of LDP with a detailed showcase with the hopes that it gets adapted more in practical scenarios. We'll first start with raw data, add noise to it, and then send it to the server for analysis. The showcase source can be found and ran at our repo on Github.
